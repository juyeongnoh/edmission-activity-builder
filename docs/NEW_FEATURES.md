# 추가 제안 기능

## 체크리스트 <-> Notion DB 동기화

![Image](https://github.com/user-attachments/assets/d83b0a42-5d5b-4138-a3d3-e9db6e2438eb)

### 개요

학생들이 기존에 사용하고 있는 앱에 체크리스트 기능이 이미 탑재되어 있어, 학생들이 자신만의 입시 플랜 데이터를 통합해서 관리할 수 있도록 **Notion과의 통합**을 제안합니다.

학생들이 자주 사용하는 노트 앱인 노션과의 통합을 통해, EdMission 체크리스트 기능의 활용도를 높이는 방안을 구상했습니다.

이를 통해, 할 일을 어디에 저장했는지 걱정할 필요 없이 EdMission과 Notion 양쪽에서 동일한 체크리스트를 확인하고 관리할 수 있습니다.

### 기대 효과

- 학생들이 이미 익숙한 Notion 환경에서도 체크리스트를 관리할 수 있어 편의성 향상
- 기존 워크플로우를 방해하지 않고 EdMission을 자연스럽게 통합할 수 있어 사용자 유지율 증가

## Slack을 통한 입시 일정 D-day 알림

![Image](https://github.com/user-attachments/assets/25b3baa7-3ff9-452e-8ec2-add8dcb12326)

### 개요

입시 준비 과정에서 중요한 마감일을 놓치지 않도록 **Slack을 통한 자동 D-day 알림 시스템**을 제안합니다.

학생과 컨설턴트가 이미 소통하고 있는 Slack 채널에 입시 일정 알림을 자동으로 전송하여, 중요한 날짜를 효과적으로 관리하고 팀 전체가 동일한 타임라인을 공유할 수 있습니다.

### 기대 효과

- 중요한 날짜를 놓치지 않고 체계적으로 관리
- 이미 사용 중인 Slack을 활용해 추가 앱 설치 불필요
